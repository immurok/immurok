# using-imk skill

Lives at [`../../imk-skill/`](../../imk-skill/) in this monorepo and is
synced to its own GitHub mirror at <https://github.com/immurok/imk-skill>
via `scripts/sync-github.sh`.

The canonical SKILL.md is at `imk-skill/skills/using-imk/SKILL.md`.

End-user install (Claude Code):

```
/plugin marketplace add immurok/imk-skill
/plugin install imk-tools@imk
```

Other agents — see `imk-skill/README.md` for per-agent install paths
(Cursor, Codex, Aider, Gemini CLI, ...).
